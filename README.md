#### 软件介绍

Word Counter能实现一个命令行文本计数统计。能正确统计导入的纯英文txt文本中的字符数，单词数，句子数，行数。

#### 使用说明

输入软件名称wc.exe
输入指令1、2、3、4、5（1——字符数；2——单词数；3——句子数；4——行数；5——注释行数）
输入需要调用与程序同目录下的txt文件名
注意：输入指令错误会提示“输入指令错误”
注意：文件名输错或不存在文件会提示“can not open file”
e.g.

wc.exe 1 1.txt
